Template Maker

Geben Sie den Namen Ihrer gewünschten Template-Extension ein.
Es wird die demotemplate-Extension kopiert und umbenannt.

Das System befindet sich im Composer-Modus. Die Extension wird nicht installiert und muss manuell bei GIT eingecheckt werden.

Keine Leerzeichen, keine Bindestriche, Keine Großbuchstaben.
